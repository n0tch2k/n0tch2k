Sublime Text 3

- 24.05.2015

A few days ago I started using the editor Sublime Text 3 for all my programming. So far I've used it for Bash scripting and HTML/CSS but it was surprisingly easy to add my own build system to handle LaTeX files. In the past days I could only scratch the surface of this very powerful editor. With all the available packages, the extendability seems endless.

For colorschemes, you can head over to Colorsublimeand grab one of their themes. Sublime Text's default theme is Monokai which is already pretty awesome though!

Yet another AUR client

- 22.05.2015

I wrote a small script called aurinstall to make it a tad more efficient to install packages from the AUR. Right now, it just covers the most important basics but it's already enough to make one's life easier.
